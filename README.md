# Ressistant
Sebuah webapp untuk membantumu mengingat _deadline-deadline_-mu

## Requirements
* Python 3.9
* Koneksi internet

## Cara Menggunakan
* Jalankan Start.bat
* Buka terminal di folder utama
* Jalankan perintah ```Activate```
* Jalankan perintah ```RunWeb```
